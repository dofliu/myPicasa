# 🚀 myPicasa 快速入門指南

歡迎使用 myPicasa！本指南將協助您快速開始使用這個美觀、功能強大的圖片與影片整理工具。

## 📥 安裝步驟

### 1️⃣ 確認環境

確保您的系統已安裝 Python 3.7 或更高版本：

```bash
python --version
```

### 2️⃣ 下載專案

```bash
git clone https://github.com/dofliu/myPicasa.git
cd myPicasa
```

### 3️⃣ 安裝依賴

```bash
pip install -r requirements.txt
```

安裝的套件包括：
- PyQt5 - GUI 框架
- Pillow - 圖片處理
- moviepy - 影片處理
- natsort - 自然排序

### 4️⃣ 啟動程式

**推薦使用現代化美化版本：**

```bash
python picasa4.py
```

## 📖 功能使用教學

### 🖼️ 功能一：圖片拼接

**目的：** 將多張圖片組合成一張網格拼接圖

**步驟：**

1. 切換到「🖼️ 圖片處理」分頁
2. 點擊「📂 選擇圖片檔案」按鈕
3. 選擇您要拼接的圖片（可多選）
4. 設定參數：
   - **列數**：例如 2（橫向有幾張）
   - **行數**：例如 2（縱向有幾張）
   - **縮放策略**：
     - 「直接縮放」- 可能變形但填滿格子
     - 「保持比例補白」- 保持原始比例，空白處補白
5. 點擊「🖼️ 拼接圖片」按鈕
6. 選擇儲存位置和檔名
7. 完成！

**範例：**
- 選擇 4 張照片
- 設定 2x2 網格
- 得到一張包含 4 張照片的拼接圖

---

### 🎞️ 功能二：GIF 動畫製作

**目的：** 將多張圖片製作成動態 GIF

**步驟：**

1. 在「🖼️ 圖片處理」分頁
2. 點擊「📂 選擇圖片檔案」
3. 選擇要製作成動畫的圖片（順序很重要！）
4. 設定「GIF 持續時間」（毫秒）
   - 500ms = 每張圖片顯示 0.5 秒
   - 1000ms = 每張圖片顯示 1 秒
5. 點擊「🎞️ 生成 GIF」按鈕
6. 選擇儲存位置
7. 完成！

**小技巧：**
- 圖片數量建議 5-20 張
- 持續時間 300-500ms 較為流暢
- 圖片檔案會按檔名自然排序

---

### 🎬 功能三：影片合併

**目的：** 將多個影片檔案合併為一個完整影片

**步驟：**

1. 切換到「🎬 影片處理」分頁
2. 點擊「📂 選擇影片檔案」按鈕
3. 選擇要合併的影片檔案（可多選）
4. 在「輸出檔名」欄位輸入檔名，例如：`完整影片.mp4`
5. 點擊「🎬 合併影片」按鈕
6. 等待處理（可能需要一些時間）
7. 完成！

**支援格式：**
- MP4, AVI, MOV, MKV, WEBM, FLV

**注意事項：**
- 影片會按檔名自然排序（video1, video2, video10...）
- 建議使用相同解析度的影片
- 處理時間取決於影片大小

---

### 🔄 功能四：圖片格式轉換

**目的：** 批次轉換圖片格式

**步驟：**

1. 切換到「🔄 格式轉換」分頁
2. 點擊「📂 選擇圖片檔案」按鈕
3. 選擇要轉換的圖片（可多選）
4. 選擇「輸出格式」（JPG, PNG, WEBP, BMP, GIF）
5. 設定「輸出資料夾」（選填）
   - 填寫 = 儲存到指定資料夾
   - 留空 = 儲存到原始檔案的資料夾
6. 點擊「✨ 開始轉換」按鈕
7. 完成！

**使用場景：**
- PNG 轉 JPG 減少檔案大小
- JPG 轉 WEBP 現代化格式
- 批次轉換大量圖片

## 🎨 主題切換

### 切換方式

點擊右上角的主題按鈕：
- 🌙 **深色模式** - 低對比度，護眼舒適，適合夜間使用

**深色主題：**
- 深灰色背景，減少眼睛疲勞
- 柔和的文字顏色
- 適合長時間使用

**淺色主題：**
- 明亮白色背景
- 高對比度，清晰易讀
- 傳統經典風格

## 💡 使用技巧

### 圖片拼接技巧

1. **選擇合適的縮放策略**
   - 照片比例相同 → 直接縮放
   - 照片比例不同 → 保持比例補白

2. **網格設定建議**
   - 4 張照片 → 2x2
   - 6 張照片 → 2x3 或 3x2
   - 9 張照片 → 3x3

3. **提升品質**
   - 使用相同尺寸的圖片
   - 預先調整圖片方向
   - 選擇高解析度圖片

### GIF 製作技巧

1. **流暢度**
   - 持續時間：300-500ms
   - 圖片數量：10-20 張最佳
   - 保持圖片尺寸一致

2. **檔案大小**
   - 減少圖片數量
   - 降低圖片解析度
   - 使用「直接縮放」策略

3. **視覺效果**
   - 圖片順序很重要
   - 考慮循環播放效果
   - 避免色彩突變

### 影片合併技巧

1. **最佳做法**
   - 使用相同解析度
   - 使用相同幀率
   - 使用相同編碼格式

2. **檔名管理**
   - 使用數字編號：video_01.mp4, video_02.mp4
   - 程式會自動自然排序
   - 避免特殊字元

3. **處理時間**
   - 小檔案（< 100MB）：幾秒鐘
   - 中檔案（100-500MB）：數分鐘
   - 大檔案（> 500MB）：可能需要 10+ 分鐘

## 🔧 常見問題解決

### Q1: 程式無法啟動？

**解決方法：**
```bash
# 檢查 Python 版本
python --version

# 重新安裝依賴
pip install -r requirements.txt --upgrade

# 嘗試啟動
python picasa4.py
```

### Q2: 圖片拼接後變形了？

**解決方法：**
選擇「保持比例補白」縮放策略，這樣可以保持原圖比例。

### Q3: GIF 動畫太大？

**解決方法：**
- 減少圖片數量
- 在拼接前縮小圖片尺寸
- 使用較低解析度的圖片

### Q4: 影片合併失敗？

**解決方法：**
- 確保所有影片格式相同
- 檢查影片編碼是否相容
- 嘗試先轉換為 MP4 格式

### Q5: 找不到檔案？

**解決方法：**
- 檢查檔案路徑是否正確
- 確認檔案沒有被其他程式佔用
- 檢查檔案權限

## 📚 進階使用

### 鍵盤快捷鍵

雖然目前沒有設定快捷鍵，但您可以：
- 使用 Tab 鍵在欄位間切換
- 使用 Enter 確認對話框

對於大量檔案，建議：
1. 分批處理（每次 20-30 個檔案）
2. 定期儲存結果
3. 注意檔案命名規則

### 最佳效能

提升處理速度：
1. 關閉其他大型程式
2. 使用 SSD 儲存裝置
3. 確保足夠的系統記憶體
