# myPicasa - 圖片與影片整理工具

一個功能豐富的桌面應用程式，提供圖片拼接、GIF 動畫製作、影片合併和圖片格式轉換等功能。

## 功能特色

### 圖片處理
- **圖片拼接** - 將多張圖片組合成網格狀拼接圖
  - 可自訂列數與行數
  - 支援兩種縮放策略：直接縮放 / 保持比例補白
  - 可調整圖片間距
- **GIF 動畫製作** - 將多張圖片轉換為 GIF 動畫
  - 可調整幀持續時間
  - 支援無限循環播放

### 影片處理
- **影片合併** - 將多個影片檔案合併為單一影片
  - 支援格式：MP4, AVI, MOV, MKV, WEBM, FLV
  - 自動自然排序（正確處理檔名中的數字順序）
  - 高品質編碼（libx264 + AAC）

### 圖片格式轉換
- **批次格式轉換** - 快速轉換圖片格式
  - 支援格式：JPG, PNG, WEBP, BMP, GIF
  - 可指定輸出資料夾
  - 批次處理多個檔案

## 系統需求

- Python 3.7+

### 1. 克隆專案


### 2. 安裝依賴套件
```bash
pip install -r requirements.txt
```

### 3. 運行應用程式
```bash
python picasa2.py
```

## 使用說明

### 圖片拼接
1. 點擊「選擇圖片檔案」按鈕
2. 選擇要拼接的圖片
3. 設定網格參數（列數、行數）
4. 選擇縮放策略
5. 點擊「拼接圖片」按鈕
6. 選擇儲存位置

### GIF 動畫製作
1. 點擊「選擇圖片檔案」按鈕
2. 選擇要製作動畫的圖片序列
3. 設定動畫持續時間（毫秒）
4. 點擊「生成 GIF 動畫」按鈕
5. 選擇儲存位置

### 影片合併
1. 切換到「影片處理」分頁
2. 點擊「選擇影片檔案」按鈕
3. 選擇要合併的影片檔案
4. 設定輸出檔名
5. 點擊「合併影片」按鈕
6. 等待處理完成

### 圖片格式轉換
1. 切換到「圖片格式轉換」分頁
2. 點擊「選擇要轉換的圖片檔案」按鈕
3. 選擇輸出格式
4. 選擇輸出資料夾（選填）
5. 點擊「開始轉換」按鈕

## 常見問題

### Q: 為什麼我的圖片拼接後變形了？
A: 請選擇「保持比例補白」的縮放策略，這樣可以保持原圖比例並自動補白。

### Q: 支援哪些影片格式？
A: 支援 MP4, AVI, MOV, MKV, WEBM, FLV 等常見格式。

### Q: GIF 動畫太大怎麼辦？
A: 可以在拼接前先縮小圖片尺寸，或減少幀數來降低檔案大小。

### Q: 影片合併時出現錯誤？
A: 請確保所有影片的編碼格式相容，建議使用相同解析度和幀率的影片。
