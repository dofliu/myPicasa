## 🌟 設計特色

### 1. 主題系統

支援深色和淺色兩種主題，可即時切換：

- **淺色主題** (預設)
  - 清新明亮的介面
  - 適合白天使用
  - 護眼舒適

- **深色主題**
  - 低對比度設計
  - 適合夜間使用
  - 減少眼睛疲勞

**切換方式：** 點擊右上角的主題切換按鈕

### 2. 配色方案

#### 深色主題配色
- 主色調：`#3B82F6` (藍色)
- 背景色：`#1E293B` (深灰)
- 表面色：`#334155` (灰色)
- 文字色：`#F1F5F9` (淺灰)

#### 淺色主題配色
- 主色調：`#3B82F6` (藍色)
- 背景色：`#F8FAFC` (淺灰)
- 表面色：`#FFFFFF` (白色)
- 文字色：`#1E293B` (深灰)

### 3. UI 元素設計

#### 按鈕設計
- **主要按鈕**
  - 藍色背景 (#3B82F6)
  - 白色文字
  - 圓角 6px
  - Hover 效果：變暗
  - 點擊效果：輕微下移

- **次要按鈕**
  - 透明背景
  - 邊框設計
  - 與主題配色一致

#### 輸入框設計
- 2px 邊框
- 圓角 6px
- Focus 時邊框變為主色調
- 內邊距 8px 12px

#### 卡片設計
- 白色/深色背景
- 圓角 12px
- 陰影效果
- 清晰的標題區

#### 分頁設計
- 圓角標籤
- 選中狀態高亮
- Hover 效果
- Emoji 圖標增強識別

### 4. 間距系統

統一的間距體系：
- 小間距：8px
- 中間距：16px
- 大間距：20px
- 元素內邊距：8-12px
- 視窗邊距：20px

### 5. 字體系統

- **主要字體：** Microsoft JhengHei, Segoe UI, Arial
- **標題大小：** 16pt (粗體)
- **正文大小：** 10pt
- **版本標籤：** 9pt

### 6. Emoji 圖標

為每個功能添加 Emoji 圖標，提升視覺識別度：

- 🎨 應用程式標題
- 📁 檔案操作
- 🖼️ 圖片處理
- 🎬 影片處理
- 🔄 格式轉換
- ⚙️ 參數設定
- ✅ 成功訊息
- ⚠️ 警告訊息
- ❌ 錯誤訊息

## 🚀 使用指南

### 啟動程式

```bash
python picasa4.py
```

### 功能操作

#### 1. 圖片拼接
1. 切換到「🖼️ 圖片處理」分頁
2. 點擊「📂 選擇圖片檔案」
3. 設定網格參數（列數、行數）
4. 選擇縮放策略
5. 點擊「🖼️ 拼接圖片」

#### 2. GIF 製作
1. 在圖片處理分頁選擇圖片
2. 設定 GIF 持續時間
3. 點擊「🎞️ 生成 GIF」

#### 3. 影片合併
1. 切換到「🎬 影片處理」分頁
2. 點擊「📂 選擇影片檔案」
3. 設定輸出檔名
4. 點擊「🎬 合併影片」

#### 4. 格式轉換
1. 切換到「🔄 格式轉換」分頁
2. 選擇要轉換的圖片
3. 選擇輸出格式
4. 設定輸出資料夾（可選）
5. 點擊「✨ 開始轉換」

#### 5. 主題切換
- 點擊右上角「🌙 深色模式」或「☀️ 淺色模式」按鈕
- 主題會即時切換

## 📐 技術實現

### QSS (Qt Style Sheets)

使用 QSS 實現樣式管理，類似 CSS：

```python
# 深色主題
stylesheet = ModernStyle.get_dark_stylesheet()
app.setStyleSheet(stylesheet)

# 淺色主題
stylesheet = ModernStyle.get_light_stylesheet()
app.setStyleSheet(stylesheet)
```

### 樣式類別

`ModernStyle` 類別提供：
- `get_dark_stylesheet()` - 深色主題樣式
- `get_light_stylesheet()` - 淺色主題樣式
- `get_card_style(theme="light")` - 卡片樣式
- `DARK_THEME` - 深色主題配色字典
- `LIGHT_THEME` - 淺色主題配色字典

### 自訂屬性

使用 Qt 自訂屬性來標記特殊元素：

```python
# 標題標籤
label.setProperty("heading", True)

# 次要按鈕
button.setProperty("secondary", True)
```

## 🔧 自訂樣式

### 修改主色調

編輯 `utils/modern_style.py`：

```python
LIGHT_THEME = {
    'primary': '#3B82F6',  # 修改這裡來改變主色調
    # ...
}
```

### 新增自訂樣式

在 `ModernStyle` 類別中新增方法：

```python
@classmethod
def get_custom_style(cls):
    return """
        /* 自訂樣式 */
        QWidget {
            /* ... */
        }
    """
```

### 調整間距

修改佈局間距：

```python
layout.setSpacing(20)  # 元素間距
layout.setContentsMargins(20, 20, 20, 20)  # 邊距
```
